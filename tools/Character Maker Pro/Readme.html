Known Bugs/Problems:

Problem: Weird spacing in help strings.
Reason: This is intentional for the time being until I've decided on the proper wording as once I make the change I cannot undo it.

Problem: Word abbreviations.
Reason: The Japanese language as we all know is very different then English and while we may need 4 spaces for the word right they would only need 2. In those types of situation, which there were quite a few I was forced to make compromises and used RT for right. Because of this limitation I've included a list of abbreviations I used and what they actually mean.

If you are a skilled HEX editor and willing to help contact me so we can fix this problem.

Frequently Asked Questions:

Q: I found a bug/translation error/untranslated material..Etc OR I have a suggestion/comment/complaint..Etc
A: Please post them in in the support topic for Character Maker Pro.

Q: The font is messed up and everything is miss-aligned!
A: Make sure the MS UI Gothic font is installed as it's required by Character Maker Pro to run properly, navigate to "C:\Windows\Fonts" and check if it’s there. If not, go into your Character Maker Pro install directory and in the "Extras" folder copy it over "msgothi0.ttc" to "C:\Windows\Fonts".
